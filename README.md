# Custom-Driver-FRC
A Custom Circuit Board for the FIRST Robotics Competition, utilizing the new SystemCore as it features 5 SL Molex Connectors for receiving digital pins, a screw terminal for power input from the Mini Power Module, a shift register + 555 timer to send signals/update them, and LED indicators to represent the different sub-systems of a robot.
